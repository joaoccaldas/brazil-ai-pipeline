O Tradutor de 'Corporativês' para o Mundo Real

Sabe aquele email que começa com "Prezados, visando maximizar nossas sinergias..."? Pois é. No Brasil, a gente chama isso de **enrolação**. Enquanto a IA "séria" está tentando escrever esses emails, eu construí uma IA para destruí-los.

A ideia aqui não é apenas rir (embora seja terapêutico), é mostrar como a IA pode ser usada para **extrair intenção**. No Sandbox de hoje, criei um mapeador de termos que traduz o "bullshit" corporativo para o que o seu chefe ou cliente realmente quer.

Como eu fiz (Sem frescura)

Usei uma lógica de mapeamento de palavras-chave. Se o sistema detecta palavras como "sinergia" ou "workflow", ele injeta a verdade nua e crua no resultado. É um exemplo simples de como processar linguagem natural focado em **utilidade emocional**.

O Código (Copie e use se tiver coragem)

Aqui está o coração do meu tradutor de gírias de escritório:

<script>
const dicionarioDaVerdade = [
    { key: 'sinergia', val: 'trabalhar mais sem bônus' },
    { key: 'asap', val: 'pra ontem pq eu esqueci' },
    { key: 'alinhamento', val: 'reunião inútil' }
];

function desvendar() {
    let input = document.getElementById('txt').value.toLowerCase();
    let verdade = dicionarioDaVerdade.find(m => input.includes(m.key));
    alert(verdade ? verdade.val : "Só enrolação mesmo.");
}
</script>
